# SynovaLab - Sitio Web

Sitio web de **SynovaLab**, consultora online especializada en calidad y seguridad alimentaria.

## 🚀 Cómo publicar en GitHub Pages

1. Sube este repositorio completo a GitHub
2. Ve a **Settings** → **Pages**
3. En **Source** selecciona la rama `main` y la carpeta `/ (root)`
4. Haz clic en **Save**
5. Tu sitio estará disponible en `https://tu-usuario.github.io/nombre-repo/`

## 📁 Estructura del proyecto

```
synovalab-web/
├── index.html                  ← Página principal
├── README.md                   ← Este archivo
├── assets/
│   ├── css/
│   │   └── styles.css          ← Estilos
│   ├── js/
│   │   └── main.js             ← Funcionalidad (carrusel, menú, animaciones)
│   └── img/
│       ├── logo.jpg            ← Logo SynovaLab
│       ├── banner.png          ← Banner principal
│       ├── cursos/             ← Imágenes de los 9 cursos
│       └── galeria/            ← Fotos de la galería
```
